<p class="preamble-title">
Homothetic Centers: X(3616)-X(3637)
</p>
<img src="rightri.svg" style="height:30px" alt="rightri">
</header>
<div class="preamble-sect">

<p>Suppose that X is a triangle center, that M
is the medial triangle, and that t is a real number. The t-dilation of
M from X, denoted by H(X; M, t), is a triangle center. If X = x : y : z
(trilinears), then</p>

<p>H(X; M, t) = bc[(1 + t)ax + (1 - t)by + (1 - t)cz] : ca[(1 - t)ax +
(1 + t)by + (1 - t)cz] : ab[(1 - t)ax + (1 - t)by + (1 + t)cz],</p>

<p>with inverse given by</p>

<p>H<sup>-1</sup>(X; M, t) = bc[(1 - t)(by + cz) - 2ax] : ca[(1 - t)(cz
+ ax) - 2by : ab[(1 - t)(ax + by) - 2cz].</p>
