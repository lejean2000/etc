<h3 id="X37861">X(37861)&nbsp;=&nbsp; X(1123)X(3086)&cap;X(2324)X(3083) </h3>
Barycentrics &nbsp;&nbsp; 8*((b+c)*a^4-2*(b+c)*(b^2+c^2)*a^2-8*b^2*c^2*a+(b^2-c^2)^2*(b+c))*S*a-(a-b-c)*(a^7+(b+c)*a^6-(3*b^2+14*b*c+3*c^2)*a^5-3*(b+c)*(b^2+6*b*c+c^2)*a^4+(b-3*c)*(3*b-c)*(b^2+6*b*c+c^2)*a^3+(b^2-c^2)*(b-c)*(3*b^2+22*b*c+3*c^2)*a^2-(b^2-c^2)^2*(b^2-6*b*c+c^2)*a-(b^2-c^2)^3*(b-c)) : :
<br>
Barycentrics &nbsp;&nbsp; (4aR^2 + abc)(4R^2 (b + c -a) + abc) + 4Rabc(8R^2 + ab + ac) + 16R^2 a^2 bc : : (Vijay, April 15, 2020)
<br>
Barycentrics &nbsp;&nbsp; S^4 + 2S^3 a(b+c)+S^2 abc(4a + b + c) + 4S a^2 b^2 c^2 + a^3 b^2 c^2(b + c - a) : : (Vijay, April 15, 2020)
<p>
See Barry Wolk, Emmanuel Garcia, and César Lozada, <a href="http://geometriadominicana.blogspot.com/2019/12/"><b>Conics intersecting the sides of a triangle</b></a> and <a href="https://groups.io/g/euclid/message/806"><b>Euclid 806 </b></a>.  They constructed two conics, one designated "inner", and the other, "outer", and they established that X(37861) and X(37862) are the centers of the inner and outer two conics, respectively.
</p>
<p>
At the suggestion of Dasari Naga Vijay Krishna (April 14, 2020), the inner conic constructed by Wolk, Garcia, and Lozada can be called the <b>Paasche conic</b>, with reference to the Paasche point, X(1123), from which the six points on the conic are constructed; see Randy Hutson's construction at X(1123).  Vijay noted that the six points are
</p>
<p>
0 : 2R : c,  &nbsp;&nbsp;&nbsp;&nbsp; 0 : b : 2R, &nbsp;&nbsp;&nbsp;&nbsp; a : 0 : 2R, &nbsp;&nbsp;&nbsp;&nbsp; 2R : 0 : c, &nbsp;&nbsp;&nbsp;&nbsp; 2R : b : 0, &nbsp;&nbsp;&nbsp;&nbsp; a : 2R : 0,
</p>
<p>
and that an equation for the Paasche conic is
</p>
<p>
2R(bcx<sup>2</sup> + cay<sup>2</sup> + abz<sup>2</sup>) - a(4R<sup>2</sup> + bc)yz - b(4R<sup>2</sup> + ca)zx  - c(4R<sup>2</sup> + ab)xy = 0.
</p>
<p>
See <a href="X(37861).pdf"><b>X(37861)</b></a>.  (Dasari Naga Vijay Krishna, April 24, 2020)
</p>
<p>
The inner Paasche conic is an ellipse for all triangles ABC. (Peter Moses, April 26, 2020)
</p>
<p>
The inner Paasche can therefore be called the <b>Paasche ellipse</b>.  The outer conic is here named the <b>outer Paasche conic</b>.  8 (April 27, 2020) has established the following properties and equations for the two conics:
</p>
<p>
The 6 points on the Paasche ellipse, with barycentrics given above as
</p>
<p>
0 : 2R : c,  &nbsp;&nbsp;&nbsp;&nbsp; 0 : b : 2R, &nbsp;&nbsp;&nbsp;&nbsp; a : 0 : 2R, &nbsp;&nbsp;&nbsp;&nbsp; 2R : 0 : c, &nbsp;&nbsp;&nbsp;&nbsp; 2R : b : 0, &nbsp;&nbsp;&nbsp;&nbsp; a : 2R : 0,
</p>
<p>
are also given, respectively, by these barycentrics:
</p>
<p>
0 : ab : S = 0 : 1 : sin C <br>
0 : S : ca = 0 : sin B : 1 <br>
S : 0 : bc = sin A : 0 : 1 <br>
ab : 0 : S = 1 : 0 : sin C <br>
ca : S : 0 = 1 : sin B : 0 <br>
S : bc : 0 = sin A : 1 : 0
</p>
<p>
The corresponding points on the outer Paasche conic, which is <b>not an ellipse for all triangles ABC</b>, are given, respectively, by
</p>
<p>
0 : ab : -S = 0 : -1 : sin C <br>
0 : -S : ca = 0 sin B : -1 <br>
-S : 0 : bc = sin A : 0 : -1 <br>
ab : 0 : -S = -1 : 0 : sin C <br>
ca : -S : 0 = -1 : sin B : 0 <br>
-S : bc : 0 = sin A : -1 : 0
</p>
<p>
Equations for the inner Paasche conic (the Paasche ellipse) and the outer Paasche conic are, respectively,
</p>
<p>
(x*y+x*z+y*z)*S^2 - (b*c*x^2+a*c*y^2+a*b*z^2)*S + (b*z+c*y)*a*b*c*x + (SB+SC)*b*c*y*z = 0;
</p>
<p>
(x*y+x*z+y*z)*S^2 + (b*c*x^2+a*c*y^2+a*b*z^2)*S + (b*z+c*y)*a*b*c*x + (SB+SC)*b*c*y*z = 0.
</p>
<p>
The definition of the Paasche conics generalizes as follows:  Let P = p : q : r and U = u : v : w (barycentrics) be triangle centers having the same degree of homogeneity in a,b,c. The <b>(p,u) generalized Paasche conic</b>, denoted by GPC(p,u), is the conic that passes through these six points:
</p>
<p>
0 : r : w<br>
0 : v : q<br>
u : 0 : p<br>
r : 0 : w<br>
q : v : 0<br>
u : p : 0<br>
</p>
<p>
An equation for GPC(p,u) follows:
</p>
<p>
p v w x^2 + q w u y^2 + r u v z^2 - (q r u + u v w) y z  - (r p v + u v w) z x - (p q w + u v w) x y = 0.
</p>
<p>
Note that, for example, GPC(p,2u) is not the same conic as GPC(p,u); i.e., the definition of generalized Paasche conic depends on the representations of centers P and U. (Clark Kimberling, May 1, 2020)
</p>
<p>
The perspector of GPC(p,u) is the barycentric quotient P/U = p v w : q w u : r u v, and the center of GPC(p,u) is
</p>
<p>
2 p^2 v^2 w^2 - u v w (2 p q r + 2 q r u + 2 p r v + 2 p q w + r u v + q u w) : :
</p>
<p>
X(37861) lies on these lines: {1123, 3086}, {2324, 3083}
</p>
