POINT</h3>
Barycentrics &nbsp;&nbsp; f(a,b,c) : f(b,c,a) : f(c,a,b),
where f(a,b,c) = a (a^2 - b^2 + b c - c^2) (a^2 (b + c) - 2 a b c - b^3
+ b^2 c + b c^2 - c^3) / ((b^2 + c^2 - a^2) (a^6 - a^4 (b^2 - b c +
c^2) - a^3 b c (b + c) - a^2 (b^4 - b^3 c - 2 b^2 c^2 - b c^3 + c^4) +
a b (b - c)^2 c (b + c) + (b - c)^4 (b + c)^2))
