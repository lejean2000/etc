<h3 id="X5625">X(5625)&nbsp;=&nbsp; MIDPOINT OF X(1) AND
X(86)</h3>
Barycentrics &nbsp;&nbsp; (2a + b + c)(a<sup>2</sup> + 2ab +
2ac + bc) : (2b + c + a)(b<sup>2</sup> + 2bc + 2ba + ca) : (2c + a +
b)(c<sup>2</sup> + 2ca + 2cb + ab)<br>
<b>X(5625) = X(1654) - 5*X(3616) </b>

<p>Suppose that P is a point in the plane of a triangle ABC. Let
L<sub>A</sub> be the line through P parallel to BC, and let
B<sub>A</sub> = L<sub>A</sub>&cap;AB and C<sub>A</sub> =
L<sub>A</sub>&cap;CA. Define A<sub>B</sub> and C<sub>B</sub>
cyclically, and define A<sub>C</sub> and B<sub>C</sub> cyclically. Let
U<sub>A</sub> be the line of the midpoints of segments
A<sub>B</sub>A<sub>C</sub> and B<sub>C</sub>C<sub>B</sub>, and define
U<sub>B</sub> and U<sub>C</sub> cyclically. The lines U<sub>A</sub>,
U<sub>B</sub>, U<sub>C</sub> concur in a point Q = Q(P). If P is given
by barycentrics p : q : r, then Q = g(p, q, r) : g(q, r, p) : g(r, p,
q), where g(p,q,r) = (2p + q + r)(p<sup>2</sup> + 2pq + 2pr + qr). If P
= X(1), then Q = X(5625). &nbsp;&nbsp; (Seiichi Kirikami, December 8,
2013)</p>

<p>X(5625) lies on these lines:
{1,75}, {10,4478}, {519,4733}, {524,551}, {726,3723}, {1100,1125}, {1255,4756}, {1279,3636}, {1654,3616}, {1961,3699}, {1962,4427}, {2796,4353}, {3244,4923}, {3624,3759}, {3842,4649}, {3945,4655}, {3993,4670}</p>

</p>
