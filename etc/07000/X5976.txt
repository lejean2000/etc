<h3 id="X5976">X(5976)&nbsp;=&nbsp; X(39)-OF-1st-ANTI-BROCARD TRIANGLE</h3>
Barycentrics&nbsp;&nbsp; f(a,b,c) : f(b,c,a) : f(c,a,b), where f(a,b,c) = (a<sup>4</sup> -
b<sup>2</sup>c<sup>2</sup>)(b<sup>4</sup> + c<sup>4</sup> -
a<sup>2</sup>b<sup>2</sup> - a<sup>2</sup>c<sup>2</sup>)
&nbsp;&nbsp;&nbsp; (Peter Moses, August 21, 2014)

<p>Barycentrics&nbsp;&nbsp; g(A,B,C) : g(B,C,A) : g(C,A,B), where
g(A,B,C) = sin B csc(B - 2&omega;) + sin C csc(C - 2&omega;)
&nbsp;&nbsp;&nbsp; (Randy Hutson, August 22, 2014)</p>

<p>The 1st anti-Brocard triangle is defined at X(5939). Suppose that P
= p : q : r (barycentrics) is a point. Then
P-of-1st-anti-Brocard-triangle is the point</p>

<p>T(P) = (a<sup>4</sup> - b<sup>2</sup>c<sup>2</sup>)p +
(c<sup>4</sup> - a<sup>2</sup>b<sup>2</sup>)q + (b<sup>4</sup> -
a<sup>2</sup>c<sup>2</sup>)r : (b<sup>4</sup> -
c<sup>2</sup>a<sup>2</sup>)q + (a<sup>4</sup> -
b<sup>2</sup>c<sup>2</sup>)r + (c<sup>4</sup> -
b<sup>2</sup>a<sup>2</sup>)p : (c<sup>4</sup> -
a<sup>2</sup>b<sup>2</sup>)r + (b<sup>4</sup> -
c<sup>2</sup>a<sup>2</sup>)p + (a<sup>4</sup> -
c<sup>2</sup>b<sup>2</sup>)q</p>

<p>The inverse mapping T*(P), satisfying T*(T(P)) = T(T*(P)) = P, is
the given by</p>

<p>T*(P) = a<sup>2</sup>p + c<sup>2</sup>q + b<sup>2</sup>r :
c<sup>2</sup>p + b<sup>2</sup>q + a<sup>2</sup>r : b<sup>2</sup>p +
a<sup>2</sup>q + c<sup>2</sup>r</p>
