<h3 id="X5390">X(5390)&nbsp;=&nbsp;EULER-MORLEY-ZHAO
POINT</h3>
Trilinears&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;T(A,B,C)
: T(B,C,A) : T(C,A,B), where T(A,B,C) = cos(B - C) - cos(B + C) -
cos(B/3 + C/3) + cos(5B/3 + 5C/3) - sin(C - B/3 - &pi;/6) - sin(B - C/3
- &pi;/6) + sin(B + 5C/3 - &pi;/6) + sin(C + 5B/3 -
&pi;/6)&nbsp;&nbsp;(Chris van Tienhoven, April 7, 2013)

<p>Barycentrics&nbsp;&nbsp; a*f(A/3, B/3, C/3) : b*f(B/3, C/3, A/3) :
c*f(C/3, A/3, B/3), where f(x,y,z) is defined using the abbreviations
[m,n] for sin(x + 2my + 2nz) + sin(x + 2ny + 2mz) and [m] for [m,m]/2,
as follows:</p>

<p>f(x,y,z) = [-3,-1] + 5[-2,-1] + [-2,0] - 5[-1,1] - 3[-1,2] - [-1,3]
+ [0,1] + 2[0,2] - [0,3] - 2[1,3] - 2[2,3] + [-3] - 2[-2] + 3[-1] +
3[0] + 3[1] + 3[2] - 2[3] - [4]&nbsp;&nbsp; (Barry Wolk)</p>

<p>Let DEF be the classical Morley triangle. The Euler lines of the
three triangles AEF, BFD, CDE to concur in X(5390), as discovered by
Zhao Yong of Anhui, China, October 2, 2012. For a construction and
derivation of barycentric coordinates by Shi Yong, see Problem 20
at
<a href="http://faculty.evansville.edu/ck6/integer/unsolved.html" class="bold">Unsolved Problems and Rewards</a>. For
further developments, including the development of trilinear and
barycentric coordinates as shown above, type X(5390) into Search at
<a href="http://tech.groups.yahoo.com/group/Hyacinthos/" style="font-weight: bold">Hyacinthos</a>.</p>
