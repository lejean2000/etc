<h3 id="X5997">X(5997)&nbsp;=&nbsp; 1st HUTSON-WOLK POINT</h3>
Barycentrics &nbsp;&nbsp; a - [(a - b + c)(a + b - c)]<sup>1/2</sup> : :

<p>X(5997) and X(5998) are described by Randy Hutson (Hyacinthos
#21045) and Barry Wolk (Hyacinthos #21047, June 1, 2012). One of the 4
points of intersection of the incircle and Steiner inellipse is a
triangle center, X(5997), and the other 3 are the vertices of a central
triangle; of those 3, let A&prime; be the one farthest from A, and define B&prime;
and C&prime; cyclically. The triangle A&prime;B&prime;C&prime; is perspective to ABC at
X(5998). Barycentrics for A&prime;, B&prime;, C&prime; follow:</p>

<p>A&prime; = a - [(a - b + c)(a + b - c)]<sup>1/2</sup> : b + [(b - c + a)(b
+ c - a)]<sup>1/2</sup> : c + [(c - a + b)(c + a -
b)]<sup>1/2</sup><br>
B&prime; = a + [(a - b + c)(a + b - c)]<sup>1/2</sup> : b - [(b - c + a)(b +
c - a)]<sup>1/2</sup> : c + [(c - a + b)(c + a - b)]<sup>1/2</sup><br>
C&prime; = a + [(a - b + c)(a + b - c)]<sup>1/2</sup> : b + [(b - c + a)(b +
c - a)]<sup>1/2</sup> : c - [(c - a + b)(c + a - b)]<sup>1/2</sup></p>

<p>X(5997) is the {X(1),X(508)}-harmonic conjugate of X(5998). (Randy
Hutson, August 22, 2014)</p>

<p>X(5997) lies on the incircle, the Steiner inellipse, and this line: {1,508)
</p>
