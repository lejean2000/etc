<h3 id="X5373">X(5373)&nbsp;=&nbsp;EQUIAREALITY CENTER</h3>
Trilinears&nbsp;&nbsp; x : y : z = f(A,B,C) : f(B,C,A) : f(C,A,B) where (x,y,z) is the solution of the following system:<br>
(y<sup>2</sup> + z<sup>2</sup>)cos A + 2yz = sin A<br>
(z<sup>2</sup> + x<sup>2</sup>)cos B + 2zx = sin B<br>
(x<sup>2</sup> + y<sup>2</sup>)cos C + 2xy = sin C, where S = 2*area(ABC).
<br>
(There is a unique solution with real x,y,z if the reference triangle ABC is acute.)
<p>
For any point X inside an acute triangle ABC, let A&prime; B&prime; C&prime; denote the pedal triangle of X. Then X(5373) is the point X for which the quadrilaterals AC&prime;XB&prime;, BA&prime;XC&prime;, CB&prime;XA&prime; all have the same area.
</p>
