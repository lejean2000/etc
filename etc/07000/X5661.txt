<h3 id="X5661">X(5661)&nbsp;=&nbsp; MINIMIZER ON BROCARD AXIS OF x<sup>2</sup> + y<sup>2</sup> + z<sup>2</sup></h3>
Barycentrics&nbsp;&nbsp; f(a,b,c) : f(b,c,a) :
f(c,a,b), where f(a,b,c) = a<sup>2</sup>(a<sup>6</sup>b<sup>4</sup> +
a<sup>6</sup>c<sup>4</sup> - 2a<sup>4</sup>b<sup>6</sup> -
2a<sup>4</sup>c<sup>6</sup> + a<sup>2</sup>b<sup>8</sup> +
a<sup>2</sup>c<sup>8</sup> - a<sup>2</sup>b<sup>6</sup>c<sup>2</sup> -
a<sup>2</sup>b<sup>2</sup>c<sup>6</sup> +
2a<sup>2</sup>b<sup>4</sup>c<sup>4</sup> + b<sup>8</sup>c<sup>2</sup> +
b<sup>2</sup>c<sup>8</sup> - b<sup>6</sup>c<sup>4</sup> -
b<sup>4</sup>c<sup>6</sup>)

<p>Suppose that P = p : q : r and U = u : v : w (homogeneous
barycentric coordinates). Let (x,y,z) be normalized barycentric
coordinates for an arbitrary point X. The point T on the line PU that
minimizes x<sup>2</sup> + y<sup>2</sup> + z<sup>2</sup> is given (Peter
Moses, June 23, 2014) by</p>

<p>T = (qu - pv)(pv + rv - qu - qw) + (ru - pw)(pw + qw - ru - rv)</p>

<p>X(5661) is the minimizer T when P = X(3) and Q = X(6), so that PU is
the Brocard axis. Other examples follow:</p>

<p>If PU = Lemoine axis (P = X(187), Q = X(237)), then T = X(3229)<br>
If PU = orthic axis (P = X(230), Q = X(223)), then T = X(230)<br>
If PU = anti-orthic axis (P = X(44), Q = X(513)), then T = X(1575)<br>
If PU = De Longchamps line (P = X(325), Q = X(523)), then T =
X(325)<br>
If PU = Gergonne line (P = X(241), Q = X(514)), then T = X(3008)<br>
If P = X(1) and Q = X(3), then T = X(5662)<br>
If P = X(1) and Q = X(6), then T = X(5701)</p>

<p>X(5661) lies on these lines: {2,647}, {3,6}, {3672,4235}</p>

<p>X(5661) = crossdifference of every pair of points on the line
X(237)X(523)<br>
X(5661) = crosssum of X(6) and X(5091)</p>
