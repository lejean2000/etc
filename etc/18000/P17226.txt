<p class="preamble-title">
Moses points: X(17226)-X(17400)
</p>
<img src="rightri.svg" style="height:30px" alt="rightri">
</header>
<div class="preamble-sect">

<p>
Consider the set triangle centers having first barycentric of the form
</p>
<p>
k(1)*a^2 + k(2)*(ab + ac) + k(3)(b^2 + c^2) + k(4)*bc,
</p>
<p>
where k(i) is in {-2,-1,0,1,2} for i = 1,2,3,4.  There are 273 such centers, of which 99 are listed above.  The remaining 174 appear below as X(17227) to X(17400), with names of the form MOSES (k(1),k(2),k(3),k(4)) POINT.  Barycentrics contributed by Peter Moses, April 8, 2018.
</p>
<p>
We refer to ordered tuples such as (k(1),k(2),k(3),k(4)) as the <b>Moses code</b> for a point X.  If X has a first barycentric that is a polynomial in a,b,c, then the Moses notation can be generalized in a straightforward manner, as in X(17669)-X(17698).  The notation can also be extended to cover certain nonpolynomials; e.g. X(615) can be coded as the Moses (1,0,0,0;-2S) point.
</p>
<p>
Specifically, for polynomial centers of degree 3, the Moses code is the 6-tuple (k(1),...,k(6)) of coefficients in
</p>
<p>
k(1)*a^3 + k(2)*a^2*(b + c) + k(3)*a*(b^2 + c^2) + k(4)*a*b*c + k(5)*(b^3 + c^3) + k(6)*(b^2*c + b*c^2); see, for example, X(17591).
</p>
<p>
For degree 5, the Moses code is the 12-tupe (k(1),...,k(12)) representing
</p>
<p>
k(1)*a^5 + k(2)*a^4*(b + c) + k(3)*a^3*(b^2 + c^2) + k(4)*a^2*b*c + k(5)*a^2*(b^3 + c^3) + k(6)*a^2*(b^2*c + b*c^2) + k(7)*a*(b^4 + c^4) + k(8)*a*(b^3 c + b*c^3) + a(9)*a*b^2*c^2 + a(10)*(b^5 + c^5) + a(11)(b^4 c + b*c^4) + a(12)*(b^3 c^2 + b^2 c^3); see for example, X(17787).
