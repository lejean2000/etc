Moses-Euler Points: X(14030)-X(14047)
</p>
<img src="rightri.svg" style="height:30px" alt="rightri">

<div class="preamble-sect">
<p>
On July 21, 2017, Peter Moses noted a relatively simple form for a point P(k) on the Euler line:
P(k) = b<sup>4</sup> - b<sup>2</sup>c<sup>2</sup> + c<sup>4</sup> + k*(a<sup>4</sup> + b<sup>2</sup>c<sup>2</sup>) : : (barycentrics)
</p>
<p>
The point is here named the <b>Moses-Euler Point (k = f)</b>, where f is a function of (a,b,c), homogeneous of degree 0.  The point is given by the combo P(k) = 3(k + 1)(a<sup>4</sup> + b<sup>4</sup> + c<sup>4</sup>)*X(2) + 4(k - 1)S<sup>2</sup>*X(4).
</p>
<p>

<p>
Every point on the Euler line is a Moses-Euler (k = f(a,b,c)) point.  A few more examples follow:
</p>
<p>
If k = (4S^2-(a^4+b^4+c^4))/(4S^2+(a^4+b^4+c^4)), then P(k) = X(3).
</p>
<p>
If k = (4S^2+(a^4+b^4+c^4))/(4S^2-(a^4+b^4+c^4)), then P(k) = X(5).
</p>
<p>
If k = (2S^2-(a^4+b^4+c^4))/(2S^2+(a^4+b^4+c^4)), then P(k) = X(20).
</p>
<p>
If k = (12S^2-(a^4+b^4+c^4))/(12S^2+(a^4+b^4+c^4)), then P(k) = X(140).
</p>
<p>
If k = ((a+b+c) (a^3-a^2 b-a b^2+b^3-a^2 c-b^2 c-a c^2-b c^2+c^3)+(a^4+b^4+c^4))/((a+b+c) (a^3-a^2 b-a b^2+b^3-a^2 c-b^2 c-a c^2-b c^2+c^3)-(a^4+b^4+c^4)), then P(k) = X(21).
</p>
<p>
If k = (a^6-a^4 b^2-a^2 b^4+b^6-a^4 c^2-2 a^2 b^2 c^2-b^4 c^2-a^2 c^4-b^2 c^4+c^6+(a^2+b^2+c^2) (a^4+b^4+c^4))/(a^6-a^4 b^2-a^2 b^4+b^6-a^4 c^2-2 a^2 b^2 c^2-b^4 c^2-a^2 c^4-b^2 c^4+c^6-(a^2+b^2+c^2) (a^4+b^4+c^4)), then P(k) = X(22).
</p>
<p>
The Shinagawa coefficients (G(a,b,c), H(a,b,c)) for the Moses-Euler point P(k) = b<sup>4</sup> - b<sup>2</sup>c<sup>2</sup> + c<sup>4</sup> + k*(a<sup>4</sup> + b<sup>2</sup>c<sup>2</sup>) : : are given by
</p>
<p>
G(a,b,c) = (k + 1)( - 1 + cot<sup>2</sup> &omega;)<br>
H(a,b,c) = 2(k - 1).
</p>

Following are combos for the Moses-Euler point P(k):
</p>
<p>
P(k) = 3 (a^4 + b^4 + c^4) (1 + k) X(2) + 4 (k - 1) S^2 X(4)<br>
P(k) = 3 (1 + k) (Cot(w)^2 - 1) X(2) + 2 (k - 1) X(4)<br>
</p>
<p>

</p>
