<div class="preamble-sect">
<br>
Centers X(616)-X(642) were contributed by Bernard Gibert, March 2, 2001. Notation:
	<p class="bold">S<sub>A</sub> = (b<sup>2</sup> +
        c<sup>2</sup> - a<sup>2</sup>)/2&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
        S<sub>B</sub> = (c<sup>2</sup> + a<sup>2</sup> -
        b<sup>2</sup>)/2&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; S<sub>C</sub> =
        (a<sup>2</sup> + b<sup>2</sup> - c<sup>2</sup>)/2</p>
	<br>
Joe Goggins (Oct. 19, 2005) found trilinears for points in this section, using the following notation. Let
        <p>F(13) = a csc(A + &pi;/3) + b csc(B + &pi;/3) + c csc(C + &pi;/3)
<br>
F(14) = a csc(A - &pi;/3) + b csc(B - &pi;/3) + c csc(C - &pi;/3)
<br>
These are based on trilinears for the isogonic centers, X(13) and X(14). In like manner, F(i) is defined for i = 15, 16, 17, 18, 61,
62.
</p><p>
Using this notation, we have, for example,<br>
X(616) = F(13)/a - 2 csc(A + &pi;/3) : F(13)/b - 2 csc(B + &pi;/3) :  F(13)/c - 2 csc(C + &pi;/3)
<br>
X(617) = F(14)/a - 2 csc(A - &pi;/3) : F(14)/b - 2 csc(B - &pi;/3) : F(14)/c - 2 csc(C - &pi;/3)
</p><p>
Trilinears of this sort are given below at X(i)  for these i: 616-619, 621-624, 627-630, and 633-636.<br>
