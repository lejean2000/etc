<p>
This preamble was contributed by Dasari Naga Vijay Krishna, May 21, 2020.<br>
<p>
In the preamble just before X(37994), six points are defined as follows:
</p>
<p>
Ab = 0 : 2R : c,  Ac = 0 : b : 2R,  Bc = a : 0 : 2R, Ba = 2R : 0 : c, Ca = 2R : b : 0, Cb = a : 2R : 0  (These points lie on the Paasche ellipse; see X(37861) and X(3788)
</p>
<p>
Here we define six more points:
</p>
<p>
A'b = 0 : 4R + c : c and A'c = 0 : b : 4R + b <br>
B'c = a : 0 : 4R + a and B'a = 4R + c : 0 : c <br>
C'a = 4R+b : b : 0 and C'b = a : 4R+a : 0 <br>
Qa = midpoint of A'bA'c, Qb = midpoint of B'cB'a, Qc = midpoint of C'aC'b
</p>
<p>
Geometrically, A'b = midpoint of B and Ab, A'c = midpoint of C and Ac, etc..  Using the notation in the preamble just before X(38310), these midpoints are given by taking (p,q,r) = (4R + a, 4R + b, 4R + c) and (u,v,w) = (a,b,c).
</p>
<p>
Define 21 points as follows:
</p>
<p>
L'a = A'cC'a&cap;A'bB'a, L'b = A'bB'a&cap;B'cC'b, L'c = B'cC'b&cap;A'cC'a <br>
P'a = C'aB'c&cap;C'bB'a, P'b = C'bA'c&cap;C'aA'b, P'c = B'aA'c&cap;B'cA'b <br>
K'a = C'bA'c&cap;A'bB'c, K'b = A'cB'a&cap;B'cC'a, K'c = B'aC'b&cap;C'aA'b <br>
Ma = C'bAc&cap;B'cAb, Mb = C'aBc&cap;A'cBa, Mc = A'bCa&cap;B'aCb <br>
Na = B'aCa&cap;C'aBa, Nb = C'bAb&cap;A'bCb, Nc = A'cBc&cap;B'cAc <br>
Va = A'cB'c&cap;A'bC'b, Vb = B'aC'a&cap;B'cA'c, Vc = A'bC'b&cap;B'aC'a <br>
Ra= B'aCb&cap;C'aBc, Rb = A'bCa&cap;C'bAc, Rc = B'cAb&cap;A'cBa <br>
</p>
<p>
First barycentrics representing the 21 points follow:
</p>
<p>
L'a = 2R (4R + b + c) : b (2R + c) : c (2R + b)  <br>
P'a = ((4R + a)(16R^2  + 4R (a + b + c) + (a b + b c + c a)) + a b c : b (4R + a)(4R + a + c) : c (4R + a)( 4R + a + b) <br>
K'a = 2R a (4R + b + c) : (4R + a)(2R + b)(4R + c): (4R + a)(2R + c)(4R + b)  <br>
Ma = a (4R^2 - b c) : 2R(4R + a)(2R + b)  : 2R(4R + a)(2R + c) <br>
Na = 12R^2 + 4R b + 4R c + b c : b (2R + c) : c (2R + b)  <br>
Va = -2R a (4R + b + c) : b (4R + a)(2R + c) : c (4R + a)(2R + b)  <br>
Ra = a^2 b c - 4R^2 (4R + b)(4R + c) : 2R a b c - 4R^2 b (4R + c) : 2R a b c - 4R^2 c (4R + b) <br>
</p>
<p>
Also, from the preamble just before X(37944),
</p>
<p>
La = 4R^2 - b c : b (2R + c) : c (2R + b) ) <br>
Pa = 16R^4 - a^2 b c : 2R b (4 R^2 - a c) : 2R c ( 4R^2 - a b) <br>
Ka = a (4R^2 - b c) : 4R^2 (2R + b) : 4R^2 (2R + c) <br>
Ha = -a (4 R^2 - b c) : 2 R b (2R + c) : 2 R c (2R + b) ) <br>
</p>
<p>
Related triangles are here given names as follows:
</p>
<p>

P'aP'bP'c = 5th Vijay-Paasche-Hutson triangle <br>
K'aK'bK'c = 6th Vijay-Paasche-Hutson triangle <br>

NaNbNc = 8th Vijay-Paasche-Hutson triangle <br>
VaVbVc = 9th Vijay-Paasche-Hutson triangle <br>
RaRbRc = 10th Vijay-Paasche-Hutson triangle <br>
QaQbQc = 11th Vijay-Paasche-Hutson triangle
</p>
<p>
Collinearities:
</p>
<p>
L'a, Hb, Hc <br>
A, Na, Va, La, L'a, Ha  <br>

P'a, L'a, K'a are collinear <br>

(Each list of collinearities represents a family of collinearities; e.g., the list L'a, Hb, Hc also represents L'b, Hc, Ha and L'c, Ha, Hb.)
</p>
