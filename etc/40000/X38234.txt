<h3 id="X38234">X(38234) = PERSPECTOR OF THESE TRIANGLES: ABC AND VIJAY-PAASCHE REFLECTION TRIANGLE </h3>
Barycentrics &nbsp;&nbsp; a / (a S + 2 b SB +2 c SC - a b c ) :  b / (b S + 2 a SA +2 c SC - a b c ) : c / (c S + 2 a SA +2 b SB - a b c )
<p>
Let Ab = 0 : 2R : c, Ac = 0 : b : 2R, Bc = a : 0 : 2R, Ba = 2R : 0 : c, Ca = 2R : b : 0, Cb = a : 2R : 0; these are points on the Paasche ellipse, as at  X(37861) and X(37881). Let B'a and C'a be the reflections of Ba and Ca in the line BC, and define B'c, A'c, A'b, C'b cyclically. Let
</p>
<p>
Va = B'cA'c&cap;A'bC'b,&nbsp;&nbsp;&nbsp;&nbsp; Vb = C'aB'a&cap;B'cA'c,&nbsp;&nbsp;&nbsp;&nbsp; Vc = C'aB'a&cap;A'bC'b.
</p>
<p>
The triangle VaVbVc, here named the <b>Vijay-Paasch reflection triangle</b>, is perspective to ABC, and the perspector is X(38234).
