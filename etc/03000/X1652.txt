<h3 id="X1652">X(1652)&nbsp;=&nbsp;4th EVANS PERSPECTOR</h3>
<a id="4th-EvansPerspector"></a>
Trilinears&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;f(A,B,C) : f(B,C,A) : f(C,A,B), where f(A,B,C) = sqrt(3/4) - sin(A + &pi;/3) + sin(B + &pi;/3) + sin(C + &pi;/3)<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
= sqrt(3/4) - cos(A - &pi;/6) + cos(B - &pi;/6) + cos(C - &pi;/6)<br>
Trilinears&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; sqrt(3) tan(A/2) +
tan(B/2) tan(C/2) : sqrt(3) tan(B/2) + tan(C/2) tan(A/2) : sqrt(3)
tan(C/2) + tan(A/2) tan(B/2)<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
= sqrt(3) tan(A/2) + (b + c - a)/(a + b + c) : sqrt(3) tan(B/2) + (c +
a - b)/(a + b + c) : sqrt(3) tan(C/2) + (a + b - c)/(a + b +
c)&nbsp;&nbsp;&nbsp;&nbsp; (M. Iliev, 5/13/07)
<p>Barycentrics&nbsp;&nbsp;(sin A)f(A,B,C) : (sin B)f(B,C,A) : (sin
C)f(C,A,B)</p>
