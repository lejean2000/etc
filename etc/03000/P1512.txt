<p class="preamble-title">
Orthojoins: X(1512)-X(1568)
</p>
<img src="rightri.svg" style="height:30px" alt="rightri">
</header>
<div class="preamble-sect">

<br>
The orthojoin of a point X = x : y : z other than X(6) is defined in
the Glossary algebraically in terms of variables a,b,c. When these are
sidelengths of a triangle, orthojoin(X) is the orthopole of the
trilinear polar of the isogonal conjugate of X. (Added to ETC 6/18/03.)
Let
<p>D(a,b,c) = bc[2abcx + c<sup>3</sup>y + b<sup>3</sup>z - bc(by + cz)
- a<sup>2</sup>(cy + bz)],<br>
E(a,b,c) = [(a<sup>4</sup> - (b<sup>2</sup> -
c<sup>2</sup>)<sup>2</sup>]x - 4a<sup>2</sup>bc(y cos B + z cos C),<br>
f(a,b,c) = D(a,b,c)E(a,b,c).</p>

<p>Then <b>orthojoin(X) = f(a,b,c) : f(b,c,a) : f(c,a,b).</b> Below,
orthojoin(X) is written as H(X).</p>

<p>Suppose X is not X(2) and does not lie on a sideline of triangle
ABC. The crossdifference of X and X(2) has first trilinear<br>
a(b-c)x', where x' = (by - cz)/(b-c).</p>

<p>Let X <sup>-1</sup> denote the isogonal conjugate of X. Then<br>
H(X <sup>-1</sup>) = (by - cz) cos A : (cz - ax) cos B : (ax - by) cos
C.</p>

<p>In other words, if X lies on a line PG through the centroid G, then
H(X <sup>-1</sup>) lies on the line HQ, where H denotes the orthocenter
and Q is a point that can be determined from the above formula.
Examples:</p>

<p>If X is on the Euler line, L(2,3), then H(X <sup>-1</sup>) is on the
line L(4,6);<br>
If X is on L(2,6,), then H(X <sup>-1</sup>) is on the Euler line;<br>
If X is on L(1,2), then H(X <sup>-1</sup>) is on L(4,9);<br>
If X is on L(2,7), then H(X <sup>-1</sup>) is on L(1,4).</p>

<p>Suppose P is not X(6), and let<br>

<p>Let X be a point on line PX(6) and not on a sideline of ABC. Then
H(X <sup>-1</sup>) is on line S"S'. Examples:</p>

<p>If X is a center on L(1,6) and X is not X(6), then H(X
<sup>-1</sup>) is on L(119,517).<br>
If X is a center on L(2,6) and X is not X(6), then H(X <sup>-1</sup>)
is on L(114,511).<br>
If X is a center on L(3,6) and X is not X(6), then H(X <sup>-1</sup>)
is on L(113,30).<br>
If X is a center on L(6,31) and X is not X(6), then H(X <sup>-1</sup>)
is on L(118,516).<br>
If X is a center on L(6,44) and X is not X(6), then H(X <sup>-1</sup>)
is on L(117,515).</p>
